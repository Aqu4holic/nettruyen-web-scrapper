# Nettruyen web scrapper

  

  

## Introduction

  

  

A web scrapper made with Python.

  
  
  

Demo: https://youtu.be/1Z29sCHYPIc

  
  
  

## How to use

- **Windows 64bit**

    - Requirements:

        - Python with these libraries:

            - requests

            - urllib3

            - beautifulsoup4

            - opencv-python

            - Pillow

            - numpy

            - termcolor

            - colorama

            - tk

            - tqdm (optional)

    - After installing all of the above libs, compile python as usual.
